# Type-in-Games

These games and applications have been typed in from books and magazines published last century, specifically for use in my Commodore64 Ultimate.
<br />
I have modified several of the originals, to either add more modern controls (W,S,A,D) for movement and to combine the Joystick/Keyboard as one program, as well as adding a brief intro screen.<br />
Most of these are entered on my C64 Ultimate, as I absolutely love the feel and sound of that keyboard.  When entering many lines of DATA numbers, I cheated by adding an external USB numberpad to add speed and accuracy to the entries.

I've also created a Linux disk utility which is basically a python front-end to the VICE petcat and c1541 tools, allowing disk/prg edits on the fly.  When I'm happy with the tool, I'll upload it here also.
